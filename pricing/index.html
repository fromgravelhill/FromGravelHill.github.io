Project Range

For a simple custom site with several pages and light interactivity, these ranges are typical:

Scope Price Notes
Easy/Simple Design $300–$900 For low-friction builds with minimal revisions
Normal / Mid $900-$1,300 A default quote for a typical small multi-page site
Upper End (Revision Heavy) $1,300–$1,500 If there’s significant back-and-forth, extra setup help, or repeated layout changes
